Getting Started

Hello and welcome to the Jupiter help documentation. Here you will find a reference guide with the functions and requirments of the Jupiter Data Management System.

To get started, here are some definitions of terms used in the system.

Template: A template is the blueprint for building a collection of data. Templates are typically named for the type of data to be collected. Examples include: Medical Records, Administrative Files, and more. Templates are set up for you by the system administrator.

Record: A record is a template that has had data input. If you have the medical record for George Washington that is a record. So is the medical record for John Adams. They are the same template but examples of two records. There are three tabs on a record: Info-The specific data [Fields] for search and information, Notes-General information relating to the record, and Documents-Uploaded files related to the record.

Field: A field is a type of data value on a record. "George" is the First Name field for the George Washington record. Fields can be either text (numbers and letters) or dates. Fields (like templates) are set up for you by the system administrator.

Document: A document is a file that has been associated with a record. This file can be of many different types (image, pdf, text file). A list of the acceptable file types can be found in the "Documents" section of the help files.


Making a New Record

The primary purpose of this system is to store and retrieve data about a wide variety of things. In this section, we will go over how to store data in Jupiter.

To create a new record, follow these steps:
1. Click on "New Record" at the top of the screen.
2. Select the template from the list on the left of the screen.
3. Click "New Record"

After clicking "New Record" you should see the Info section for the record you selected. You add data such as "First Name" to the Info tab.

Note 1: You do not have to fill out all of the Fields on the Note section to save the record.

Note 2: You MUST have at least one Field on the Info tab filled out otherwise the system will not allow you to save.
The Notes tab allows you to enter information without being confined to a Field type. You can add notes to a new record before saving.

It is important to note that NO DOCUMENTS CAN BE UPLOADED TO A RECORD BEFORE SAVING THE FIRST TIME. The Upload tab will not be available to you to use when a record has not been saved yet.

Once you have as much data as you can filled out, click the SAVE button above the record. This saves the record for retrieval later on.

Finding a Record

Once a record has been saved, you can retrieve it by following these steps:

1. Click on "Search" at the top of the screen.
2. Select the template you want to search for. This matches what the record was saved as (ex. If a Medical Record was saved then select Medical Record from the list.)
3. You will notice that fields should have appeared on the left of your screen. You can enter data here and either click SEARCH or the Enter key on your keyboard to search.

Note 1: You can enter as much information as you know in the search fields. The search will take all data and find a record matching what you put in. For example, if you put in First Name: "George" and Last Name: "Washington" you will bring back only that have records for George Washington (not anyone with a First Name of "George").

Note 2: The search is EXACT. This means that if you are looking for John Smith, you will not find it if you accidently misspell or use another version of the name (John / Johnathan).

Once you search, you will be presented with all records that match in the system as a list. They will display identifying information that has been previously set and a button labeled "Open". Each row will have it's own Open button. Locate the record you wish to view and click the open button. This will open a new window with the same three tabs that are in the New Record window but with the saved data from the record.

Updating a Record / Documents

In an open record you are now able to edit and add data as you see fit. You can update any incorrect or missing Field with data. You can also edit the notes associated with the record.

While a record is open you can now add documents to it. To add a document, you must do the following:

1. Click on the Documents tab on the open record.
2. Enter a Description of the document you are uploading. A short explanation that will help you later works the best.
3. Click on the "Browse" button to locate the file you wish to upload.
4. Once the record is saved, the document will be uploaded to the system.

Note: Once a document is attached to a record, it will show up in the list labeled "Attached Docs" in blue on the Documents tab. This will give you the short description and a button to open the document. SOME FILES WILL OPEN TO VIEW OTHERS WILL DOWNLOAD TO YOUR MACHINE TO BE OPENED OUTSIDE OF THE SYSTEM

If you know a document has been attached to a record but it is not showing up in the "Attached Docs" list, click the "Refresh" button to repopulate the list.
